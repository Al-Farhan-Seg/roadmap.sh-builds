# Accessible Form UI

[View the project requirements](https://roadmap.sh/projects/accessible-form-ui)

In this project, we will create an accessible form interface using only HTML and CSS.

The form will include fields for:

- Full name
- Email address
- Password
- Password confirmation
- A button for toggling password visibility

The interface will also include a profile-completeness indicator and a checklist of requirements needed to reach 100% completion.

This version of the form is a static UI component and is not fully functional. JavaScript can be added later to provide validation, password visibility controls, and dynamic progress updates.

The goal of this project is to practise HTML and CSS while creating a form that is accessible and easy to use, including for people who use assistive technologies.
