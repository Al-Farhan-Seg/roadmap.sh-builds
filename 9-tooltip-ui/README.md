# Tooltip UI

[View the project requirements](https://roadmap.sh/projects/tooltip-ui)

In this project, we will create navigation tooltips using only HTML and CSS.

The interface will include:

* Navigation links
* Tooltips that appear when users hover over navigation items
* Clear and readable tooltip text
* Smooth tooltip transitions
* Proper positioning relative to each navigation item
* Visual indicators that connect each tooltip to its related link

The project uses CSS positioning, pseudo-elements, hover states, and transitions to create the tooltip behaviour without JavaScript.

The goal of this project is to practise CSS positioning and interactive states while learning how to provide additional information without cluttering the interface.
