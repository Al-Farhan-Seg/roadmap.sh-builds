# Tabs

[View the project requirements](https://roadmap.sh/projects/simple-tabs)

In this project, we will create a simple and interactive tabs component using HTML, CSS, and JavaScript.

The component will include:

* Multiple tab buttons
* Separate content panels for each tab
* An active state that shows the currently selected tab
* Dynamic switching between tab content
* Clear visual feedback for user interaction
* A layout that works across different screen sizes

JavaScript is used to detect tab clicks, update the active tab, hide inactive content panels, and display the content associated with the selected tab.


The goal of this project is to practise DOM manipulation, event handling, CSS state styling, and the separation of structure, presentation, and behaviour in a small interactive component.
