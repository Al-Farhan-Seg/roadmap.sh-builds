# Age Calculator

[View the project requirements](https://roadmap.sh/projects/age-calculator)

In this project, we create an age calculator using HTML, CSS, and JavaScript.

The application includes:

* A date-of-birth input
* A button for starting the age calculation
* Validation for missing or invalid dates
* Calculation of the user's age
* A clear display of the calculated result
* A responsive and accessible interface

JavaScript handles the submitted date, compares it with the current date, and calculates the user's age while accounting for whether their birthday has already occurred during the current year.

The goal of this project is to practise working with JavaScript dates, form input, event handling, validation, conditional logic, and dynamic DOM updates.
