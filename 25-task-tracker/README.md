# Task Tracker

[View the project requirements](https://roadmap.sh/projects/task-tracker-js)

In this project, we create an interactive task tracker with a to-do list using HTML, CSS, and JavaScript.

The application includes:

* A form for adding new tasks
* A list of created tasks
* Controls for marking tasks as complete
* The ability to delete tasks
* Clear visual differences between active and completed tasks
* Input validation for empty task submissions
* Dynamic updates without reloading the page
* A responsive and accessible interface

JavaScript manages the task data, handles user interactions, creates and removes task elements, and updates task completion states dynamically.

The goal of this project is to practise DOM manipulation, event handling, form processing, arrays, application state, conditional styling, and the development of a practical interactive web application.
