# Blog Post Page

[View the project requirements](https://roadmap.sh/projects/blog-post-page)

In this project, we build a semantic blog post page using HTML and CSS.

The page includes:

* A clear article title
* Author and publication information
* Structured headings and paragraphs
* Relevant internal and external links
* Emphasised text where appropriate
* A semantic figure with an image and caption
* Accessible and readable content layout
* Responsive styling for different screen sizes

The blog post uses semantic HTML elements to give the content a meaningful structure and make it easier for browsers, search engines, and assistive technologies to understand.

The goal of this project is to practise semantic HTML, text formatting, accessible links, image presentation, content hierarchy, and the creation of readable long-form web content.
