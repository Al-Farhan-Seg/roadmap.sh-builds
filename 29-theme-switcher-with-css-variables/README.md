# Theme Switcher with CSS Variables

[View the project requirements](https://roadmap.sh/projects/theme-switcher)

In this project, we build a theme switcher that changes the appearance of a card using HTML, CSS, and custom properties.

The page includes:

* A reusable card component
* Multiple selectable themes
* Theme-specific background, text, border, and accent colours
* CSS custom properties for managing design values
* Clear visual feedback for the selected theme
* Smooth transitions between theme styles
* A responsive and accessible layout

CSS variables store the theme values and make it possible to update several visual properties from one central place. The selected theme changes the values used by the card without requiring repeated styling rules.

The goal of this project is to practise CSS custom properties, theme management, reusable styling, state-based selectors, visual consistency, and maintainable CSS architecture.
