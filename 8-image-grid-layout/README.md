# Image Grid Layout

[View the project requirements](https://roadmap.sh/projects/image-grid)

In this project, we will create a responsive image grid layout using HTML and CSS.

The layout will include:

- Multiple images arranged in a structured grid
- Images of different sizes and aspect ratios
- Consistent spacing between images
- Responsive behaviour across different screen sizes
- Proper image cropping and positioning

The project primarily uses CSS Grid to control the placement, size, and alignment of the images.

The goal of this project is to practise CSS Grid while learning how to create responsive and visually balanced image galleries.
